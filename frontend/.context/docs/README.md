## Repository Structure at a Glance
- `src/app/`: routes, layouts, and page components.
- `src/components/`: Reusable UI components (Logic & Presentation).
- `src/services/`: Backend API interaction services.
- `src/hooks/`: Custom React hooks for shared logic.
- `src/lib/`: Utilities, constants, and global types.
- `tests/`: Integrated and unit test suites.
- `public/`: Static assets (images, fonts).
